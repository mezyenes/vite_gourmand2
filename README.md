# 🍽️ Vite & Gourmand

Site web de restaurant développé en PHP avec une architecture MVC.

---

## 🚀 Description

Vite & Gourmand est un site permettant de consulter le menu, passer des commandes et laisser des avis.  
Il contient aussi un espace admin et employé.

---

## ⚙️ Fonctionnalités

- Affichage du menu  
- Commandes en ligne  
- Avis clients  
- Espace employé  
- Espace admin  
- Page contact  
- Pages légales  

---

## 🛠️ Technologies

- PHP (MVC)  
- HTML / CSS / JavaScript  
- MySQL / MongoDB  
- Composer

## ⚙️ Installation

Installer les dépendances :  
composer install  

Lancer le serveur :  
php -S localhost:8000 -t public
